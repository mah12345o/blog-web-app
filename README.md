# Blog Web App

A modern blog application built with Next.js, featuring article browsing and interactive comments.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Features

- Browse blog articles
- Comment system with emoji ratings
- Responsive design
- Dynamic content loading

## Tech Stack

- Next.js (App Router)
- TypeScript
- React Icons
- Tailwind CSS

## Project Structure

- `/app` - Application pages and components
- `/app/components` - Reusable UI components
- `/app/const` - Constants and data
- `/public` - Static assets (images, icons)
